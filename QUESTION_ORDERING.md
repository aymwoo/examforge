# 试题排序功能说明

## 功能实现

已成功实现试题按识别先后顺序排列的功能：

### 1. 数据库更改
- 在 `Question` 模型中添加了 `importOrder` 字段
- 该字段记录试题在导入时的顺序（从1开始）

### 2. 导入逻辑更新
- **Excel导入**：按行号顺序设置 `importOrder`
- **PDF视觉模式**：按AI识别返回的顺序设置 `importOrder`  
- **PDF文本模式**：按处理顺序设置 `importOrder`
- **图片文件导入**：按处理顺序设置 `importOrder`

### 3. 查询排序更新
- 问题列表查询现在优先按 `importOrder` 升序排列
- 对于没有 `importOrder` 的旧数据，仍按 `createdAt` 降序排列

### 4. 排序规则
```sql
ORDER BY importOrder ASC, createdAt DESC
```

这确保了：
1. 有导入顺序的试题按识别顺序排列（1, 2, 3...）
2. 没有导入顺序的试题按创建时间倒序排列
3. 新导入的试题总是按正确的识别顺序显示

### 5. 兼容性
- 对现有数据完全兼容
- 旧的试题（`importOrder` 为 null）仍正常显示
- 新导入的试题会自动按识别顺序排列

## 使用效果

现在当你导入PDF或其他格式的试题时，试题会严格按照AI识别的先后顺序在题库中显示，而不是按创建时间排序。
